Персонаж может крафтить предметы с использованием [[Лор/Технологии/Репликатор\|репликатора]]. Для этого ему нужно иметь три вещи: 
1. Чертёж (может храниться в электронном виде на [[Предметы/Снаряжение/Связь/E-paper планшет\|планшете]] или [[Предметы/Снаряжение/Связь/Data Seed\|Data Seed]]).
2. [[Предметы/Снаряжение/Энергетика/Power Peel\|Энергия]] для крафта:

| Размер крафтящегося предмета | Стоимость в PP |
| ---------------------------- | -------------- |
| Tiny                         | 1              |
| Small                        | 5              |
| Medium                       | 20             |
| Big                          | 60             |
| Long                         | 60             |
3. Материалы (металл, органика и пластик).
Крафт занимает столько же минут, сколько тратит PP.
