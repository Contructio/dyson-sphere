Действие — это единица активности персонажа в ход. У каждого персонажа есть столько действий, сколько у него [[Механики/Бой/Конечности\|конечностей]]. В каждом действии участвует одна или больше конечностей. Например, выстрел из пистолета использует голову и руку (так как для выстрела из пистолета требуется смотреть на цель), а выстрел из второго пистолета только руку. 

Примеры

| Действие               | Конечности    |     |
| ---------------------- | ------------- | --- |
| Ходьба                 | Обе ноги      |     |
| Выстрел из пистолета   | Голова, рука  |     |
| Выстрел из винтовки    | Голова, руки  |     |
| Удушение               | Обе руки      |     |
| Задержка дыхания       | Торс          |     |
| Карабкание по лестнице | Руки, ноги    |     |
| Перезарядка            | Обе руки      |     |
| [[Механики/Бой/Ближний бой\|Удар]]  | Нога или рука |     |
| Оглядывание            | Голова        |     |
Движение — действие, использующее обе ноги. При этом персонаж может переместиться на расстояние до пяти метров (включая прыжки для преодоления препятствий) Альтернатива: прыжок на одной ноге (перемещение до 1 метра, действие одной ногой).
