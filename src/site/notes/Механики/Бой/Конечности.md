Каждая свободная конечность даёт персонажу одно [[Механики/Бой/Действие\|действие]] за ход.

Двуручное оружие требует для атаки обе руки. При использовании двуручного оружия персонаж не может в том же ходу атаковать другой рукой или использовать её для других действий (кроме отпускания оружия).

Если конечность занята (держит предмет) — она может использовать действие только для использования этого предмета (включая удар фонариком). Иначе персонаж автоматически роняет предмет.




Каждое последующее действие в ход получает нарастающий штраф:

| Действие | Штраф |
| -------- | ----- |
| 1        | 0     |
| 2        | -3    |
| 3        | -6    |
| 4        | -9    |
| 5        | -12   |


Штраф применяется ко всем броскам, связанным с этим действием.

Навык [[Механики/Персонаж/Навыки/Multitasking\|Multitasking]] уменьшает штраф на величину, равную уровню навыка (не суммарный бонус, а именно ранг).

Пример: [[Механики/Персонаж/Навыки/Multitasking\|Multitasking]] 4, персонаж совершает третье действие.
Штраф = –6 + 4 = –2.
