Каждая свободная конечность даёт персонажу одно [[Механики/Бой/Действие\|действие]] за ход.

Двуручное оружие требует для атаки обе руки. При использовании двуручного оружия персонаж не может в том же ходу атаковать другой рукой или использовать её для других действий (кроме отпускания оружия).

Если конечность занята (держит предмет) — она может использовать действие только для использования этого предмета (включая удар фонариком). Иначе персонаж автоматически роняет предмет.




Каждое последующее действие в ход получает нарастающий штраф:

| Действие | Штраф |
| -------- | ----- |
| 1        | 0     |
| 2        | -3    |
| 3        | -6    |
| 4        | -9    |
| 5        | -12   |


Штраф применяется ко всем броскам, связанным с этим действием.

Навык [[Механики/Характеристики персонажа/Навыки/Multitasking\|Multitasking]] уменьшает штраф на величину, равную уровню навыка (не суммарный бонус, а именно ранг).

Пример: [[Механики/Характеристики персонажа/Навыки/Multitasking\|Multitasking]] 4, персонаж совершает третье действие.
Штраф = –6 + 4 = –2.

### Здоровье и восстановление

#### HP конечностей
HP всех конечностей равняется 10 + [[Механики/Характеристики персонажа/Resolve\|Resolve]]. Если в результате повышения [[Механики/Характеристики персонажа/Resolve\|Resolve]] повышается максимальный HP, то текущий HP повышается на аналогичное количество.

#### Выбивание конечностей

Когда HP конечности достигает 0, она считается выбитой и больше не даёт действия. Выбитая конечность всё ещё может получать дополнительные раны.

Когда HP головы достигает 0, персонаж не умирает мгновенно. Правила спасения описаны в разделе «Здоровье и восстановление».

#### Восстановление HP
Поспав 8 часов, персонаж восстанавливает 5 HP в каждой конечности. Если прервать данный процесс, восстанавливается только та часть HP, которая успела восстановиться:

| Времени во сне, часов | Восстановление HP |
| --------------------- | ----------------- |
| 1-2                   | 1                 |
| 3-4                   | 2                 |
| 5-6                   | 3                 |
| 7                     | 4                 |
| 8                     | 5                 |
После восьмичасового сна персонаж не может уснуть, пока снова не устанет.
