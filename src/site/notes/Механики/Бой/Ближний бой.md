Ближний бой может происходить только на дистанции CQB (см. секцию Дистанция в разделе [[Механики/Бой/Дальний бой\|Дальний бой]]). 


#### Атака
1. Бросок атаки: 1d10 + [[Механики/Персонаж/Навыки/Melee Combat\|Melee Combat]] + [[Механики/Персонаж/Характеристики/Agility\|Agility]]
2. Сложность определяется противостоящим броском [[Механики/Персонаж/Навыки/Acrobatics\|Acrobatics]].
3. При попадании бросьте урон.

#### Определение места попадания

Если бросок атаки превысил бросок уклонения на 5 или более, то игрок выбирает [[Механики/Бой/Конечности\|конечность]], куда был нанесён удар. Иначе, конечность, куда попала атака, определяется броском 1d10:

| d10  | Конечность  |
| ---- | ----------- |
| 1    | Голова      |
| 2-4  | Торс        |
| 5    | Левая рука  |
| 6    | Правая рука |
| 7-8  | Левая нога  |
| 9-10 | Правая нога |

#### Защита в ближнем бою
При первой атаке в ближнем бою по персонажу в ход он совершает один бросок [[Механики/Персонаж/Навыки/Acrobatics\|Acrobatics]]. Результат становится сложностью (DC) для всех атак ближнего боя по этому персонажу до начала его следующего хода.

· Если атакующий бросает атаку ниже или равно DC → промах. 
· Если выше → попадание.

Персонаж может уклоняться даже если уже потратил все свои действия в этом ходу.

Укрытия в ближнем бою не применяются.

### Особые приёмы в рукопашном бою

Помимо стандартной атаки (удар рукой или ногой), персонаж может использовать следующие приёмы. Каждый приём требует действия соответствующей конечностью и подчиняется правилам штрафов за множественные действия.

#### Захват

Действие: одна или две руки. 
Персонаж может захватить противника, если находится с ним в дистанции CQB.

Эффект:
· Мобильность захваченного противника становится 0 (не может перемещаться).
· Захватывающий контролирует столько рук противника, сколько использовал для захвата (одна рука = одна рука противника под контролем, две руки = две руки под контролем).
· Контролируемые руки не могут использоваться для действий.
· Освободиться из захвата можно действием захваченной конечности (контест [[Механики/Персонаж/Навыки/Melee Combat\|Melee Combat]]).

Захват длится, пока захватывающий его удерживает (тратит действие конечности(ей) на поддержание захвата в каждом ходу) или пока противник не освободится.

#### Подсечка

Действие: одна нога.

Персонаж может сделать подсечку противнику, находясь с ним в дистанции CQB.

Бросок: [[Механики/Персонаж/Навыки/Melee Combat\|Melee Combat]] против [[Механики/Персонаж/Навыки/Acrobatics\|Acrobatics]] противника.

Эффект при успехе: противник падает на землю.

Эффекты падения:
· [[Механики/Бой/Дальний бой\|Дальний бой]] (не CQB): выстрелы по лежачему противнику получают штраф –3 к броску атаки.
· [[Механики/Бой/Дальний бой\|Дальний бой]] (CQB): выстрелы по лежачему противнику получают бонус +5 к броску атаки.
· Ближний бой: атакующий получает преимущество на усмотрение мастера.

Подняться с земли — действие всех четырёх конечностей. Альтернативно персонаж может сделать бросок [[Механики/Персонаж/Навыки/Acrobatics\|акробатики]] со сложностью 15 и встать, используя действие торса.
